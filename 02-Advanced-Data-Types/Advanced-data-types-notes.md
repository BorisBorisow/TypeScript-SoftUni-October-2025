## Advanced Data Types

### Unions

    - можем да ползваме общите properties and elements.
    - означава едното или другото

### Intersection types

    - combine multiple types in one
    - has all props of the combined types

### Type Aliases

    - a custom name for a type, declared with the type keywords, can target
    - Primitives
    - Objects
    - Advanced

### keyof

    - връща само ключовете на дадения тип

### Interfaces

    - Defined by using keywords interface
    - often called duck typing or structural
    - makeSound()
